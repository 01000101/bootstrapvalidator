Validate a credit card number.

Option Equivalent HTML attribute Type Description
message data-bv-creditcard-message String The error message

From v0.3.2, in addition to using the Luhn algorithm, the validator also validate the IIN ranges and length of credit card number.

It supports validating the following cards:

  • American Express
  • Diners Club
  • Diners Club (US)
  • Discover
  • JCB
  • Laser
  • Maestro
  • Mastercard
  • Solo
  • Unionpay
  • Visa
