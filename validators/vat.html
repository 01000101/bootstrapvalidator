Validators / vat

Edit on Github

Validate VAT number. Support 32 countries.

Option Equivalent HTML attribute Type Description
message data-bv-vat-message String The error message
country data-bv-vat-country String A ISO 3166 country code. If it is not defined, it will be parsed from the VAT number.
