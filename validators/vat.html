Validate VAT number in 33 countries.

Option Equivalent HTML attribute Type Description
message data-bv-vat-message String The error message
country data-bv-vat-country String A ISO 3166 country code. If it is not defined, it will be parse from the VAT number.

The validator supports the following countries (click the example VAT number to validate it):

$(document).ready(function() {
    $('#vatForm').bootstrapValidator({

            valid: 'glyphicon glyphicon-ok',
            invalid: 'glyphicon glyphicon-remove',

        fields: {
            vat: {
                validators: {
