Validate a zip code by a country code.

Option Equivalent HTML attribute Type Description
message data-bv-zipcode-message String The error message
country data-bv-zipcode-country String A ISO 3166 country code

Currently, the validator supports the following countries:

Code Country
US United State
CA Canada
DK Denmark
