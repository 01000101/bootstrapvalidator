A jQuery plugin to validate form fields.
Designed to use with Bootstrap 3

Download v{{ site.current_version }}

Features

Code

The source code is written from scratch.
Very solid and clean

Validator

25 built-in validators and counting!
Easy to write new validator

Friendly

Show feedback icons
based on field validity

Ajax

Support Ajax
in both validator and form submission

Flexible

Field validators can be
enabled/disabled on the fly

Compatibility

Play nice with other plugins
such as Date Picker, Datetime Picker, Select2, Raty, etc.

What's news?

  • #107: Add container option for each field to indicate where the error messages are shown
  • #5: Add ip validator, support both IPv4 and IPv6
  • #6: Add isbn validator, support both ISBN 10 and ISBN 13
  • #7: Add step validator
  • #95: Add mac address validator
  • #96: Add base64 validator
  • #97: Add cvv number validator
  • #100: Add phone number validator, support US phone number only
  • #112: creditCard validator now also validates both IIN ranges and length
