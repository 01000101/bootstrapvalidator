Adding dynamic field

When working with complex form, the fields might be added to (or remove from) the form dynamically. The newly added fields also need to be validated.

This example demonstrates a sample scenario where you have to solve validating dynamic fields problem.

Before going to the details, there are some methods and events you need to know:

Method/Event Description
addField() method Adding new field
removeField() method Removing given field
added.field.bv event Called after adding new field
removed.field.bv event Called after removing given field

Assuming that you are managing a survey which consists of a question and some options that user can choose from. The form allows you to add more options but the number of options can't exceed 5.

Step 1: Writing form

Initially, our form has two fields for entering the question and option as following:

{% highlight javascript hl_lines="9 16" %} $(document).ready(function() { $('#surveyStep1Form').bootstrapValidator({ feedbackIcons: { valid: 'glyphicon glyphicon-ok', invalid: 'glyphicon glyphicon-remove', validating: 'glyphicon glyphicon-refresh' }, fields: { question: { validators: { notEmpty: { message: 'The question required and cannot be empty' } } }, 'option[]': { validators: { notEmpty: { message: 'The option required and cannot be empty' }, stringLength: { max: 100, message: 'The option must be less than 100 characters long' } } } } }); }); {% endhighlight %}

There is nothing special here, expect naming the option field as name="option[]" since the form might have more than one option.

Step 2: Adding field

Next, when clicking the button, we have to add new option to the form. To do that, I simply pre-create a option field template, then clone it after user click the button.

{% highlight javascript hl_lines="8 19 22 27" %} $(document).ready(function() { $('#surveyStep2Form') .bootstrapValidator({ ... }) // Add button click handler .on('click', '.addButton', function() { var $template = $('#surveyStep2Form').find('.optionTemplate'), $clone = $template .clone() .removeClass('hide optionTemplate') .insertBefore($template), $option = $clone .find('input') .attr('name', 'option[]'); // Add new option field $('#surveyStep2Form').bootstrapValidator('addField', $option); }) .on('added.field.bv', function(e, data) { // data.field --> The field name // data.element --> The new field element // data.options --> The new field options if (data.field === 'option[]') { // If the number of options exceed 5 if ($('#surveyStep2Form').find('[name="option[]"]').length >= 5) { // Disable the Add button $('#surveyStep2Form').find('.addButton').attr('disabled', 'disabled'); } } }); }); {% endhighlight %}

Step 3: Removing field

Our last mission are that removing the field when user click the button.

Similar to the step 2, this time we will use removeField() method and removed.field.bv to handle removing field from the form.
